# Projet Le Blog de Batman

### Cloner le projet

```
(commande commander)
git clone 
https://github.com/sophieDubois/leblogdebatman.git
(github code)
```

### Déplacer le terminal dans le dossier cloné
```
cd leblogdebatman
```

### Installer les vendors (pour recréer le dossier vendor)
```
composer install
```

### Lancer le serveur
```
symfony serve
```
